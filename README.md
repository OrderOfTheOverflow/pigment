# pigment

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Windows Dev
- add `node_modules\electron\dist\electron.exe` to Start Menu to 'activate' notifications in dev mode
